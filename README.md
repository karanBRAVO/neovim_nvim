# Neovim - Nvim Config

This is my `neovim-nvim` config.

## Steps

1. Install `neovim`

    ```bash
    sudo apt install neovim
    ```

2. verify installation

    ```bash
    nvim
    ```

3. create the config dir

    ```bash
    mkdir ~/.config/nvim/
    ```

4. clone this repo

    ```bash
    git clone https://github.com/karanBRAVO/neovim_nvim.git
    ```

5. remove git folder

    ```bash
    cd neovim_nvim
    rm -rf .git/
    ```

6. paste the config

    ```bash
    mv * .* ~/.config/nvim/
    ```
